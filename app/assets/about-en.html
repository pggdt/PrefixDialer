Overview

Add prefix number before you calling. It meets your needs when you want dail numbers with prefix numbers.

Function

  • You can dail numbers by input numbers, pick from call log or contects list.
  • It can correct the number you dialed with prefix numbers.

Known Issue

  • When finished a call from contact list, the call log doesn't refresh automaticly. If you want to see the last call log, you'd better restart this app.
  • It can't record call duration.
